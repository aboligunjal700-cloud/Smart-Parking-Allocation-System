# Smart-Parking-Allocation-System

## Project Overview:
This project is a Smart Parking System that helps find the nearest available parking slot using BFS (Breadth First Search) algorithm.

## Features:
- Take Car Details
- Finds nearest parking slot
- Displays movement path (UP, DOWN, LEFT, RIGHT)

## Algorithm used:
Breadth First Search (BFS) is used to find the shortest path to the nearest empty parking slot.

## Technologies Used:
- Python
- Data Structures ( Queue using deque )

## How to Run:
1) Install Python
2) Run the file:
   python main.py
